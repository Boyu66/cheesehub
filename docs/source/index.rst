CHEESE: Cyber Human Ecosystem of Engaged Security Education
===========================================================

CHEESE (Cyber Human Ecosystem of Engaged Security Education) is an NSF-funded
initiative to develop a learning ecosystem for cybersecurity education. The
project consists of the following:
 
* Easy to access, dynamic, web-based learning platform
* Community-contributed catalog of cybersecurity training materials 
* Community-driven platform to request, develop, share, evaluate and learn about
  content
* Learning ecosystem that is continuously updated
